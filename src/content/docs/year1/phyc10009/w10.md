# Spectra and Radiation

- $m_p$ proton mass
- $m_n$ neutron mass
- $Z$ atomic number
  - number of protons in a nucleus
- $A$ atomic mass number
  - number of nucleons
- $N$ neutron number
  - number of neutrons
  - $Z+N=A$
- $_Z^AX$
- isotopes: nuclei that contain the same number of protons but different numebrs of neutrons
